# 我的捏捏图鉴 (Squishy Log) 🧸

一个可爱的 PWA (Progressive Web App)，专门用于记录你的捏捏收藏，支持前后对比图、店铺记录和手感描述。

## 功能特点

- ✨ **捏捏档案**：记录店铺、模具、价格、克重、手感等详细信息。
- 📸 **前后对比**：支持上传捏之前、开捏中、捏之后的照片，并提供大图预览。
- 📊 **数据统计**：自动统计收纳数量、花费排行和店铺偏好。
- 🎨 **图片编辑**：内置简单的图片编辑器，支持滤镜和文字贴纸。
- 📱 **移动端优先**：专为手机设计的 UI/UX，操作顺滑。

## 快速开始

### 1. 安装依赖

```bash
npm install
```

### 2. 启动开发服务器

```bash
npm run dev
```

### 3. 构建生产版本

```bash
npm run build
```

## 技术栈

- React + TypeScript
- Vite
- Tailwind CSS (via CDN)
- Lucide React (Icons)
- LocalStorage (数据持久化)
